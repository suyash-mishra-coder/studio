# **App Name**: Mockview AI

## Core Features:

- Pre-Interview Configuration: Collect role, experience level, technical specialty, interview topic, and target company preferences via a dynamic form. These configurations will guide the interview simulation.
- AI-Powered Interview Simulation: Conduct a 45-minute real-time interview simulation with an AI-driven interviewer persona, who will ask a series of branching questions (behavioral and technical). The AI interviewer will use the configurations specified by the user in the Pre-Interview Configuration form as a tool to generate questions of appropriate type and difficulty, tailored to the stated preferences.
- Real-time Collaboration: Enable video/audio input and a shared code editor within the interview simulation environment.
- Automated Feedback Generation: Generate an instant performance score (1-10), identify strengths and weaknesses, and provide three personalized improvement tips immediately after the interview simulation. Provide a replay of the session
- Performance Dashboard: Allow users to view their session history, track their progress over time, and review past feedback to identify areas for improvement.
- Firebase Integration: Store user data, interview history, and feedback within Firebase.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey professionalism and trust.
- Background color: Light gray (#EEEEEE) for a clean and neutral backdrop.
- Accent color: Bright green (#4CAF50) for positive feedback highlights and calls to action.
- Body text and Headlines: Use 'Inter', a sans-serif font known for its neutral and modern appearance. Its clean design ensures readability across various screen sizes and contexts. Alternatively use Space Grotesk for headlines and Inter for body.
- Use minimalist, professional icons to represent various features and functionalities.
- Design a clean, intuitive layout with clear navigation and a focus on usability. Prioritize ease of use for both interview simulation and performance review.
- Incorporate subtle animations to enhance user engagement and provide feedback on interactions.
